# DBBOT
NLP-to-SQL,基于大模型和 RAG 的智能问数系统,通过自然语言查询数据库的工具，用户可以用日常语言提问，系统自动转换为SQL查询，返回可视化结果，Text-to-SQL。

DBBOT 问数是一款基于自然语言处理的数据查询工具，用户可以用日常语言提问，系统自动转换为SQL查询，返回可视化结果。彻底打通制造业数据库与AI的无缝对接的落地产品。 
DBBOT产品网站：https://nbkern.com/ai-questions
公司主要研发AI相关产品，官网https://www.nbkern.com
